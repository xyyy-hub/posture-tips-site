## üöÄ Quick Start

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

Visit [http://localhost:3000](http://localhost:3000) to view the site.

## üìù Writing New Articles

1. Check `KEYWORD_MAPPING_MASTER.md` for keyword assignment
2. Use outlines from `posturetips-content-outlines.md` as templates
3. Copy page structure from `ARTICLE_PAGE_TEMPLATE.md`
4. Follow SEO/AEO guidelines from the implementation briefs
5. Follow brand guidelines from `posturetips-brand-specs.md`
